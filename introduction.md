# quickhybrid

快速混合开发框架，JSBridge提供核心Android、iOS功能。多平台API支撑，部分兼容浏览器

包括三大部分：

- JS（前端）

- Android原生

- iOS原生

## JS（前端）

大致实现内容

```js
ES6 + Rollup + Eslint + Karma + Mocha
```
