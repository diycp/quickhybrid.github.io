# 快速开始

## 前端项目

### GIT

```js
git clone git://github.com/quickhybrid/quickhybrid.git
```

### 运行

```js
npm install

npm run build

npm run dev
```

### 其它

如可以运行单元测试，自己修改源码定制等

大致实现内容

```js
ES6 + Rollup + Eslint + Karma + Mocha
```
